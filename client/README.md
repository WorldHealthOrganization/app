# who_app

## Getting Started

Follow flutter installation instructions [here](https://flutter.dev/docs/get-started/install).

Clone the repo then:

```
cd client
flutter run
```

### iOS

```
cd $(git rev-parse --show-toplevel)
cd client
flutter pub get

cd ios
sudo gem install cocoapods   # if needed
pod install
open Runner.xcworkspace
```
